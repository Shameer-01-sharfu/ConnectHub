# ConnectHub

A modern social media platform built using Django.

Features

✅ Authentication
✅ User Profiles
✅ Follow / Unfollow
✅ Posts
✅ Likes
✅ Comments
✅ Stories
✅ Reels
✅ One-to-One Chat
✅ Reply Messages
✅ Typing Indicator
✅ Read Receipts
✅ Notifications
✅ Search

Tech Stack

Python
Django
SQLite
HTML
CSS
Bootstrap
JavaScript

How to Run

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
